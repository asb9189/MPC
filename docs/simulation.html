simulation module¶

simulation.py creates p number of parties (mininet hosts) with an extra host being created and treated as the centeral server. All nodes are directly connected to this centeral server and once all connections are establed start_protocol() is called.

simulation.ping_all()¶

Tests connectivity for each node pinging node -> server and server -> node.

simulation.start_protocol()¶

After generate.py is executed the server is started up. After sleeping for 5 seconds each node executes summation.py -H[Hostname] -P[number of parties]. We then wait for all party members to finish computation before ending the simulation printing both party and server output.
